# Comparative-Analysis-of-Spam-Phishing
## Overview

Scams have become increasingly sophisticated with the rise of AI—ranging from phishing emails and fake websites to highly personalized scam messages. This project aims to **analyze scam trends over time** and **build a machine learning model that detects scam messages (phishing/spam)**.

By combining multiple real-world datasets, we create a unified system that can both:

* Track how scams are evolving
* Automatically classify suspicious messages


## Problem:

People receive scam texts and emails every day, but many don’t know how to identify them. Traditional detection systems struggle with newer, AI-generated scams.

This project solves two key problems: 
1. **Understanding scam trends** (how scams change over time)
2. **Detecting scam messages** using Natural Language Processing (NLP)


## Target:

* Everyday users receiving suspicious messages
* Consumer protection organizations
* Researchers studying fraud trends

## We used...

This project integrates multiple datasets into a single cleaned dataset:

### 1. Phishing Email Datasets

* Multiple CSV files with varying formats:

  * sender, receiver, subject, body, URLs, label
  * subject, body, label
  * combined text + label

### 2. SMS Spam Dataset

* Message text labeled as spam or not spam


## Future Analysis will include...

* Time series analysis of scam trends
* Clustering emerging scam types
* Feature engineering (URL patterns, message length, keywords)

## This Matters because...

Scam detection is a real-world problem affecting millions of people. This project goes beyond a simple assignment by building a system that could realistically help users identify and avoid scams.
